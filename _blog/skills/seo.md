# Skill: SEO por post

## Checklist SEO antes de publicar

### Frontmatter
- [ ] `title`: 50-60 caracteres exactos, keyword principal en las primeras 3 palabras
- [ ] `description`: 140-160 caracteres, incluye keyword + beneficio concreto al lector

### Contenido
- [ ] Keyword principal aparece en el primer párrafo (natural, no forzado)
- [ ] Al menos un `h2` contiene la keyword o una variante semántica
- [ ] Longitud mínima: 800 palabras (menos no rankea bien en técnico)
- [ ] Al menos **1 link interno** a otro post o a `/#projects` / `/#about`
- [ ] Al menos **1 link externo** a fuente autoritativa: OWASP, RFC, CVE, NIST

### URL
- El nombre del archivo define la URL: `_posts/2026-05-21-mi-titulo.md` → `/2026/05/21/mi-titulo`
- Usar guiones, minúsculas, sin caracteres especiales ni tildes
- Incluir la keyword principal en el nombre del archivo

### Imagen (cuando se agregue soporte)
- `alt` descriptivo con keyword
- Formato WebP, max 100KB
